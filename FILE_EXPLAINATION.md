# cascade_*.txt
级联文件，格式为
<id>    <root_user_id>  <publish_time>  <observation path length>   <edges> <labels>

| item    | meaning |
|-------------|---------------|
| id | 信息id |
| root user id | 根用户id |
| public time | 根用户发布时间 |
| observation path length | 允许观测的传播路径长度 |
| edges | 观察到的连边 |
| labels | 预测标签 |
